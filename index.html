The game was created for training account.The game has several levels.Each level has different speed and range which increase with each level.
After pressing the button "play" you will see the numbers which you have to add and after this level you have to give the answer.
If the answer is carectly, the next level will be start.
If the answer is wrong, you will be able to play again, but from the first level.
